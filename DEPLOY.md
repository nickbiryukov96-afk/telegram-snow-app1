# Инструкция по деплою Telegram Mini App

## Быстрый деплой на Vercel

### Способ 1: Через веб-интерфейс Vercel

1. Перейдите на https://vercel.com
2. Войдите через GitHub (или создайте аккаунт)
3. Нажмите "Add New Project"
4. Перетащите папку `dist/` из проекта
5. Или подключите GitHub репозиторий
6. В настройках проекта:
   - **Build Command**: оставьте пустым (файлы уже собраны)
   - **Output Directory**: `dist`
   - **Install Command**: оставьте пустым
7. Нажмите "Deploy"
8. После деплоя получите URL вида: `https://your-project.vercel.app`

### Способ 2: Через Vercel CLI

```bash
# Установите Vercel CLI
npm install -g vercel

# Войдите в аккаунт
vercel login

# Задеплойте папку dist
cd dist
vercel --prod
```

## Альтернативные хостинги

### Netlify

1. Перейдите на https://netlify.com
2. Перетащите папку `dist/` в область "Deploy"
3. Получите URL вида: `https://your-project.netlify.app`

### GitHub Pages

1. Создайте репозиторий на GitHub
2. Загрузите содержимое папки `dist/` в репозиторий
3. Включите GitHub Pages в настройках репозитория
4. Выберите папку `dist` как source
5. Получите URL вида: `https://username.github.io/repo-name`

## Локальное тестирование через туннель

### Использование localtunnel

```bash
# Установите localtunnel
npm install -g localtunnel

# Запустите dev сервер
npm run dev

# В другом терминале создайте туннель
lt --port 5173
```

Используйте полученный HTTPS URL для настройки в BotFather.

### Использование ngrok

```bash
# Установите ngrok
brew install ngrok  # macOS
# или скачайте с https://ngrok.com

# Запустите dev сервер
npm run dev

# В другом терминале создайте туннель
ngrok http 5173
```

Используйте HTTPS URL из ngrok для настройки в BotFather.

## Настройка в BotFather

После получения URL:

1. Откройте [@BotFather](https://t.me/BotFather)
2. Отправьте `/myapps`
3. Выберите ваше приложение
4. Выберите "Edit" → "Web App URL"
5. Введите ваш URL (например: `https://your-project.vercel.app`)
6. Сохраните изменения

## Тестирование

1. Откройте вашего бота в Telegram
2. Нажмите на кнопку меню (три линии)
3. Найдите кнопку с вашим Web App
4. Нажмите на неё - приложение откроется в полноэкранном режиме

## Важные замечания

- ✅ URL должен быть HTTPS (обязательно для Telegram)
- ✅ Домен должен быть публично доступен
- ✅ Файл `index.html` должен быть в корне
- ✅ Все пути к ресурсам должны быть относительными или абсолютными

## Проверка работоспособности

После деплоя проверьте:

1. Откройте URL в браузере - должно открыться приложение
2. Проверьте консоль браузера на ошибки
3. Убедитесь, что все изображения загружаются
4. Проверьте работу в Telegram

## Обновление приложения

После изменений в коде:

1. Пересоберите проект: `npm run build`
2. Загрузите новые файлы из `dist/` на хостинг
3. Изменения применятся автоматически (или после перезапуска деплоя)
