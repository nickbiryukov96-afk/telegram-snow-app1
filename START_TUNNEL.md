# Правильный запуск туннеля

## Проблема 503 решена!

### Правильная последовательность:

1. **Остановите все процессы:**
   ```bash
   pkill -f vite
   pkill -f cloudflared
   ```

2. **Запустите dev сервер с доступом извне:**
   ```bash
   cd /Users/nikbiryukov/telegram-snow-app
   npm run dev:host
   ```
   
   Или если команды нет:
   ```bash
   npm run dev -- --host
   ```

3. **Дождитесь, пока сервер запустится** (увидите "ready in xxx ms")

4. **В другом терминале запустите туннель:**
   ```bash
   cloudflared tunnel --url http://localhost:5173
   ```

5. **Скопируйте URL** из cloudflared

6. **Проверьте в браузере** - откройте URL из cloudflared

7. **Используйте в BotFather**

## Важно:

- Dev сервер должен быть запущен **ДО** запуска туннеля
- Используйте `--host` флаг для доступа извне
- Туннель работает только пока оба процесса запущены
